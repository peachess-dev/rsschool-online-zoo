# Online-zoo [see live ](https://peachess-dev.github.io/rsschool-online-zoo/pages/main/index.html) :point_left:
This is an implementation of my school project during the study in [RS School](https://rs.school/) free-of-charge and community-based education program conducted by The Rolling Scopes developer community since 2013.

Task requirements (full description can be found [here](https://github.com/rolling-scopes-school/js-fe-course-en/blob/main/tasks/online-zoo/online-zoo.md)) 
-
- pixel perfect 
- adaptive layout from 320px wide to the largest screens
- browsers compatiblity: Google Chrome, Mozilla Firefox
- additional functionality includes menu, slider, pagination, and popup

Key skills
-
- semantic HTML5
- CSS3/ preprocessors
- JavaScript
